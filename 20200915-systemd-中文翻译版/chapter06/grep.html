6.10. Grep-3.4

Grep 软件包包含在文件内容中进行搜索的程序。

估计构建时间: 0.2 SBU
需要硬盘空间: 26 MB

6.10.1. 安装 Grep

准备编译 Grep： 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --bindir=/bin

编译该软件包： 

make

安装该软件包： 

make DESTDIR=$LFS install

该软件包的详细信息可以在第 8.33.2 节 “Grep 的内容”中找到。
