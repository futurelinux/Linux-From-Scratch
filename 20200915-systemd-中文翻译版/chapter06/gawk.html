6.9. Gawk-5.1.0

Gawk 软件包包含操作文本文件的程序。

估计构建时间: 0.2 SBU
需要硬盘空间: 46 MB

6.9.1. 安装 Gawk

首先，确保不要安装一些没有必要的文件： 

sed -i 's/extras//' Makefile.in

准备编译 Gawk： 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(./config.guess)

编译该软件包： 

make

安装该软件包： 

make DESTDIR=$LFS install

该软件包的详细信息可以在第 8.55.2 节 “Gawk 的内容”中找到。
