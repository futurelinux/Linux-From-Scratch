6.13. Patch-2.7.6

Patch 软件包包含通过应用 “补丁” 文件，修改或创建文件的程序，补丁文件通常是 diff 程序创建的。

估计构建时间: 0.1 SBU
需要硬盘空间: 13 MB

6.13.1. 安装 Patch

准备编译 Patch： 

./configure --prefix=/usr   \
            --host=$LFS_TGT \
            --build=$(build-aux/config.guess)

编译该软件包： 

make

安装该软件包： 

make DESTDIR=$LFS install

该软件包的详细信息可以在第 8.65.2 节 “Patch 的内容”中找到。
