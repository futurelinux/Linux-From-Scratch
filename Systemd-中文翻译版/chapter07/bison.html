7.9. Bison-3.7.2

Bison 软件包包含语法分析器生成器。

估计构建时间: 0.3 SBU
需要硬盘空间: 52 MB

7.9.1. 安装 Bison

准备编译 Bison： 

./configure --prefix=/usr \
            --docdir=/usr/share/doc/bison-3.7.2

配置选项的含义：

--docdir=/usr/share/doc/bison-3.7.2

该选项告诉构建系统将 Bison 文档安装到带有版本号的目录中。

编译该软件包： 

make

安装该软件包： 

make install

该软件包的详细信息可以在第 8.32.2 节 “Bison 的内容”中找到。
